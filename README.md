# Blinkit-Analytics-Dashboard-PowerBI
Power BI Dashboard analyzing Blinkit KPI's, analyzing Growth Trends along with Overall Business Performance 
**Problem Statement- Analyze Blinkit Sales & Overall Growth Performance in order to identify Key Growth Trends in Sales and Profitablity on the basis of different Parameters such as Outlet Location, Size, Type & different Grocery Item Types.**
Tools Used- Power BI, DAX, Excel & Power Query for Data Cleaning
